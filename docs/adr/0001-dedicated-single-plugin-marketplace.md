# 専用の単一プラグイン・マーケットプレイスにする

このリポジトリを、複数の自作スキルを束ねる汎用の個人マーケットプレイスにはせず、
`yamaha-network-docs` プラグイン専用のマーケットプレイスとする。repo = marketplace =
plugin = skill をすべて同名 `yamaha-network-docs` に揃えることで、利用者の導入が
`/plugin marketplace add akchan/yamaha-network-docs` →（プラグイン名が一意なので）
`/plugin install yamaha-network-docs` の2行で完結し、名前の重複・取り違えが起きない。

## 考慮した代替案

**汎用個人マーケットプレイス**（1リポジトリに grill-me 等の他スキルも束ねる）。利用者が
1回 `marketplace add` すれば複数プラグインを選べる利点があるが、(a) マーケットプレイス名と
個々のプラグイン名が乖離して導入手順の説明が増える、(b) 無関係なヤマハ以外のスキルが
同梱され用途が曖昧になる、ため見送った。将来の他スキルはそれぞれ別リポジトリで配布する。

## 帰結

公開後にこの方針を覆して汎用マーケットプレイスへ統合すると、マーケットプレイス名が変わり、
既に `add` / `install` 済みの利用者の設定が壊れる。そのため公開前に確定させる必要があった
（変更困難なため本 ADR に記録）。
